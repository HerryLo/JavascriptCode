## gulp

first install node;

## stack
* gulp-clean-css //压缩css

* gulp-uglify //混淆代码

* gulp-imagemin //压缩图片

* gulp-htmlmin //压缩html

* gulp-rename //重新命名

* gulp-file-include //模版file

* gulp-babel //babel编译

* gulp-strip-comments //删除注释

* gulp-sourcemaps //sourcemap

* gulp-postcss //处理css

* autoprefixer //浏览器样式兼容

* postcss-pxtorem //px转rem


### install
```
$ npm install gulp

$ npm install

$ gulp
```
